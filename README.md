# Text Generation using GPT-2

## Project Overview
This project demonstrates text generation using the GPT-2 language model from Hugging Face Transformers. The model is loaded, trained on a custom Artificial Intelligence dataset, and used to generate meaningful text outputs.

## Objective
To understand how Large Language Models (LLMs) work and implement a basic text generation system using GPT-2.

## Technologies Used
- Python
- Google Colab
- Hugging Face Transformers
- PyTorch
- Datasets Library

## Dataset
A custom dataset containing 41 lines related to:
- Artificial Intelligence
- Machine Learning
- Deep Learning
- Natural Language Processing
- Data Science

## Steps Performed
1. Installed required libraries.
2. Loaded GPT-2 model and tokenizer.
3. Created a custom AI dataset.
4. Loaded dataset using Hugging Face Datasets.
5. Tokenized the dataset.
6. Fine-tuned GPT-2 on the custom dataset.
7. Generated text outputs using the trained model.

## Sample Outputs

### Output 1
Artificial Intelligence Can Learn from Us

### Output 2
Artificial Intelligence, Finances and Social Systems.

### Output 3
Artificial Intelligence (AI) has become an increasingly popular choice for modern applications.

## Results
The GPT-2 model successfully generated meaningful text related to Artificial Intelligence after training on the custom dataset.

## Future Improvements
- Use a larger dataset.
- Train for more epochs.
- Deploy as a web application.
- Fine-tune larger language models such as GPT-Neo or LLaMA.
